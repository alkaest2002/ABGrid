AB-Grid

Software per la generazione di report

Benvenuto in AB-Grid, un'applicazione web per l'analisi delle dinamiche relazionali, basata sulle tecniche sociometriche di J. L. Moreno e sulla social network analysis.

Per utilizzare correttamente l'applicazione, assicurati che i dati raccolti siano memorizzati in formato YAML. Prima di iniziare, ti consigliamo di consultare il manuale d'uso (che puoi scaricare qui).

L'applicazione è stata sviluppata dal Dr. Pierpaolo Calanna e dal Dr. Gaetano Buonaiuto ed è distribuita come software open-source sotto licenza MIT. Il codice sorgente è disponibile all'indirizzo: https://github.com/alkaest2002/abgrid.

Buon lavoro.
