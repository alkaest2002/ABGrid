AB-Grid

Software per la generazione di report

Benvenuto in AB-Grid, un'applicazione web per l'analisi delle dinamiche relazionali nei gruppi orizzontali, basata sulle tecniche sociometriche di J. L. Moreno e sulla social network analysis.

Prima di iniziare, ti consigliamo di consultare il manuale d'uso.

Buon lavoro.
